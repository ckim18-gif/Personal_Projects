PROJECT RONIN v0.1.1

실행:
1. ZIP 압축을 풉니다.
2. index.html을 더블클릭합니다.
3. WASD 또는 방향키로 이동합니다.
4. 마우스를 클릭하면 공격 애니메이션이 재생됩니다.
5. R을 누르면 시작 위치로 돌아갑니다.

포함된 실제 에셋:
- FREE Samurai 2D Pixel Art v1.2: Idle / Run / Attack
- Mystic Woods Free 2.2: grass / plains / trees
- Fantasy version 2.0: forest resource props

이 빌드는 외부 라이브러리나 서버가 필요하지 않습니다.
